## What's Ahlan Commerce?

A guided onboarding book for fresh new-commer who know Rust basics. A micro app that simulate daily tasks and decisions during the development process.

## What you expect to learn from Ahlan?

- Hands on experience on implementing a complete roduction-ready app.
- How to split features into micro services.
- How to write rust-lang code in best practices.
- Learn tokio and axum
- Learn Postgresql with atlas migrtions and SQL-first approach.
- Cache design decisions and strategies with Redis.
- How to use DB per service pattern.
- How to use TDD approach for development.
- How to use CI/CD pipelines and automate your workflow.
- How to use Git and GitHub effectively.
- How to write documentation for your code.
